HTML Notes

Comments

Comments in HTML start with a <!-- and end with a -->

Descriptive Tags

  • Include main, header, footer, nav, video, article, section and others.
  • Give descriptive structure to HTML
  • Make it easier to read
  • Help with SEO optimization
  • Help with Assessibility

Image Tags

  • <img src="https://www.your-image-source.com/your-image.jpg">
  • Are self-closing
  • Must have an alt attribute
  • Us and empty alt attribute for purely decorative images

Anchor Tags

External Links
<a href="https://freecodecamp.org">this links to freecodecamp.org</a>
Internal Links
  • Create internal links that jump to different sections within a webpage
  • Assign a link's href attribute to hash symbol (#) plus the value of the id attribute of the element to which you are internally linking
  • id uniquely describes an element in a webpage
example <a href="#contacts-header">Contacts</a> ... <h2 id="contacts-header">Contacts</h2>
