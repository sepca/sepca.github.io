HTML Notes

Comments

Comments in HTML start with a <!-- and end with a -->

Descriptive Tags

  • Include main, header, footer, nav, video, article, section and others.
  • Give descriptive structure to HTML
  • Make it easier to read
  • Help with SEO optimization
  • Help with Assessibility

Image Tags

  • <img src="https://www.your-image-source.com/your-image.jpg">
  • Are self-closing
  • Must have an alt attribute
  • Us and empty alt attribute for purely decorative images
